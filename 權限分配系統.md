# 權限分配系統 操作手冊

權限分配系統提供具 **管理權（root 管理者）** 的人員，分配各 **職務／各模組** 的操作權限，並可一鍵 **還原職務的預設權限**。

> **注意**：本系統在現行校務系統的選單名稱為「**處室權限控管**」（舊稱「權限分配系統」）。功能與操作相同，本手冊以現行畫面說明。

## 模組架構與權限說明

權限分配系統分為三個頁籤（功能）：

1. **職務權限分配**：以「職務／人員」為主，設定該對象在各模組的權限。
2. **模組權限分配**：以「模組」為主，設定各職務在該模組的權限。
3. **權限預設值**：將指定職務的權限 **還原為系統預設值**。

**權限等級（每個模組對每個職務可擇一設定）**：

| 等級 | 說明 |
| --- | --- |
| 停用 | 該模組對此對象不開放。 |
| 瀏覽 | 僅能瀏覽資料。 |
| 進階瀏覽 | 可瀏覽較完整的資料。 |
| 編輯 | 可編輯／異動資料。 |
| 管理 | 可使用該模組的所有功能（最高權限）。 |

- **管理權**：root 管理者權限，可針對職務、模組做權限分配，以及還原職務預設權限。
- 各頁籤右上角的「**預設權限表**」可下載查詢各職務的預設權限。

## 一、職務權限分配

以「**處室 → 職務（或人員）**」為單位，設定其在各模組的權限。

![職務權限分配](https://iili.io/CurEFdg.png)

操作步驟：

1. 「**檢視身份**」選擇 **教師** 或 **志工**。
2. 在 **處室下拉** 選擇要設定的處室（如校長室）。
3. 選擇對象：
   - 點 **職稱**（如「校長」）＝設定 **該職稱底下所有人員**，調整後此職稱全部人員一同變動。
   - 點 **人員姓名**＝只針對 **該名人員** 設定；設定後該姓名旁會出現閃爍紅字「**特殊權限**」，代表此人權限有單獨調整過（不隨職稱變動）。
4. 在右側對 **各模組** 點選權限等級（停用／瀏覽／進階瀏覽／編輯／管理）。
5. 按右上「**儲存設定**」存檔。

> 可點右上「**預設權限表**」下載查詢各職務的預設權限。

## 二、模組權限分配

以「**模組**」為單位，設定各職務在該模組的權限（與職務權限分配相反方向）。

![模組權限分配](https://iili.io/CurkJBs.png)

操作步驟：

1. 左側「**請選擇模組**」點選一個模組（如學籍管理）。
2. 右側對 **各職稱** 點選該模組的權限等級（停用／瀏覽／進階瀏覽／編輯／管理）。
3. 按右上「**儲存設定**」存檔。

## 三、權限預設值

將指定職務的權限 **還原為系統預設值**。

![權限預設值](https://iili.io/CurDgLJ.png)

操作步驟：

1. 在「**請勾選還原職務**」勾選要還原的對象：可按「**全選**」，或依 **處室／職務** 個別勾選。
2. 按「**確定**」，所勾選職務的權限即還原為系統預設值。

## 注意事項

- 本系統屬 **管理權（root）** 功能，一般帳號看不到、也無法使用。
- 點「人員姓名」單獨調整權限後，該姓名旁會顯示紅字「**特殊權限**」做為提醒。
- 還原預設權限（權限預設值）會覆蓋該職務目前的設定，請確認後再執行。
