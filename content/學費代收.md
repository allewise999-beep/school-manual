## 一、模組簡介

「【新】學費代收」提供學校費用代收管理功能，包含費用單製作、繳費狀況查詢、現金代收銷帳及相關報表列印，支援行庫超商代收與現金代收。

模組分為三大功能區（❶ 繳費列表主視窗、❷ 銷帳紀錄快速資訊、❸ 子標籤切換）。

![](https://iili.io/CCuAVX1.png)

| 功能標籤 | 說明 |
|---------|------|
| 繳費列表 | 查詢及管理所有費用單，並可查看學生繳費狀況 |
| 活動繳費 | 集合多個費用項目一次向學生收費 |
| 管理設定 | 設定電子繳費單、表單簽章及代收人員 |

> **權限說明**：管理員可進行全部操作；系統指定的代收人員可使用現金代收功能；家長可透過 APP 查詢並線上繳費（需啟用電子繳費單）。

## 二、繳費列表

繳費列表為模組主視窗，顯示所有已建立的費用單清單，可依條件篩選並進入各費用單進行操作。

{% tabs %}
{% tab title="學生繳費單據列表" %}

顯示所有費用單清單，包含單據名稱、APP 開放狀態、日期範圍、適用年級及繳費人數統計。

操作步驟：

1. 進入模組後預設顯示「**學生繳費單據列表**」。
2. 可使用「**快速查詢**」下拉選單篩選特定條件的費用單。
3. 點選任一費用單的「**單據名稱**」進入該費用單的操作畫面。
4. 點右上角「**新增**」可建立新的費用單。

{% endtab %}

{% tab title="學生個別查詢" %}

依學號或姓名查詢特定學生的所有費用繳納狀況。

操作步驟：

1. 點選「**學生個別查詢**」頁籤。
2. 在搜尋欄輸入學號或姓名，點「**查詢**」顯示該生的繳費記錄。
3. 可查看各費用單的繳費狀態、繳費日期及繳費管道。

## 三、費用單操作

點選費用單後進入操作頁面，左側顯示工作流程按鈕（❶ 繳費項目與匯入製單、❷ 費用單詳情、❸ 產生單據至銷帳流程），右側顯示費用單基本資料。

![](https://iili.io/CCuRYQt.png)

### 工作流程說明

| 步驟 | 功能 | 說明 |
|------|------|------|
| 1 | 繳費項目 | 設定各收費項目名稱及金額（含減免設定） |
| 2 | 匯入製單 | 將學籍系統的學生名單匯入並套用收費項目 |
| 3 | 產生單據 | 正式產生含條碼的繳費單據 |
| 4 | 繳費修改 | 手動調整個別學生的金額或備註 |
| 5 | 列印繳費單 | 印出學生的繳費通知單 |
| 6 | 繳費查詢 | 查詢各班學生繳費狀態 |
| 7 | 查詢銷帳 | 查看銀行或超商回傳的銷帳記錄 |
| 8 | 校內現金代收 | 學生到校繳費時由代收人員操作 |
| 9 | 列印結帳報表 | 列印各式統計報表 |

> ⚠️ 操作順序：繳費項目設定 → 匯入製單 → 產生單據 → 列印繳費單 → 收款/銷帳。**學生繳費後不得再執行「產生單據」。**

## 四、新增費用單

點選「**新增**」建立新費用單，填寫基本設定（❶ 單據名稱與類別、❷ 日期與銀行帳號、❸ 儲存確認）。

![](https://iili.io/CCuRlGn.png)

操作步驟：

1. 填寫「**單據名稱**」（❶）（此名稱將顯示於繳費單抬頭）。
2. 選擇「**單據類別**」（如：註冊費、學雜費、午餐費、課後輔導費等）。
3. 設定「**開始日期**」與「**截止日期**」（❷）（繳費單的有效繳費期間）。
4. 設定「**可延遲日**」（截止日後仍可至行庫超商繳費的天數）。
5. 選擇「**銀行帳號**」（代收帳號）。
6. 填寫「**核准文號**」（選填，若有相關公文文號）。
7. 設定「**是否開放**」（是否開放家長線上查詢及繳費）。
8. 填寫「**備註說明**」（會顯示於繳費單備註欄）。
9. 勾選「**繳費年班**」（設定此費用單適用的年級）。
10. 點「**確定**」（❸）儲存完成建立。

> ⚠️ 請務必先在「**管理設定 → 表單簽章設定**」完成簽章設定，再進行費用單製作。未有銷帳記錄的費用單可刪除；已有銷帳記錄者無法刪除。

## 五、活動繳費

活動繳費可將多個費用項目集合為一份費用單，方便學生一次繳清多項費用（❶ 活動繳費頁籤）。

![](https://iili.io/CCuAOdv.png)

操作步驟：

1. 點「**活動繳費**」頁籤（❶）。
2. 若目前無待收合併項目，頁面顯示「目前無待收合併項目」。
3. 有項目時，可選擇要合併收取的費用項目，設定學生及金額後產生活動繳費單。

## 六、管理設定

管理設定包含三個主要設定區（❶ 電子繳費單設定、❷ 表單簽章設定、❸ 代收人員設定、❹ 學生資料匯出）。

![](https://iili.io/CCuAbh7.png)

{% tabs %}
{% tab title="1、電子繳費單設定" %}


設定是否啟用 APP 推播繳費通知及電子化繳費單。

操作步驟：

1. 點「**管理設定**」頁籤（❶）。
2. 啟用「**電子繳費單**」後，可在學生個人資料中發送 APP 推播通知。
3. 可查看全校學生數、APP 綁定學生數及電子繳費單申請學生數統計。

{% endtab %}
{% tab title="2、表單簽章設定" %}


設定各費用單上顯示的主管簽章，需依不同銀行帳號分別設定。

操作步驟：

1. 在「**表單簽章設定**」區（❷）點「**新增簽章**」。
2. 上傳簽章圖片（需為指定規格），設定對應的銀行帳號。
3. 點「**儲存**」完成設定。

> ⚠️ 請在製作費用單之前先完成簽章設定。

{% endtab %}
{% tab title="3、代收人員設定" %}


設定可在校內執行現金代收的人員。

操作步驟：

1. 在「**代收人員設定**」區（❸）點「**新增代收人員**」。
2. 選擇人員後，該人員可使用自己的帳號登入並執行現金代收作業。
3. 可進行密碼還原或刪除代收人員。

{% endtab %}
{% tab title="4、學生資料匯出" %}


將學生資料匯出，方便製作繳費名單。

操作步驟：

1. 點「**匯出資料**」（❹），選擇年級及班級。
2. 系統提供學生資料的 Excel 格式匯出，可用於匯入製單作業。

## 七、常見問題

**Q1：新增費用單後為何找不到學生？**

A：新增費用單只是建立費用單框架，需要在費用單中依序執行：設定「**繳費項目**」→「**匯入製單**」→「**產生單據**」，才會將學生名單帶入並產生個別繳費通知單。

**Q2：學生繳費後，繳費清單沒有更新？**

A：繳費資料需等待銀行每日銷帳檔傳入後才會更新。可至「**查詢銷帳**」確認銷帳記錄，或手動執行現金代收銷帳。

**Q3：為何「產生單據」後不能再修改？**

A：一旦學生完成繳費（有銷帳記錄），費用單即被鎖定以保護財務記錄的完整性。如需修改，請確認所有銷帳記錄，並聯繫系統管理員協助。

**Q4：如何讓家長透過 APP 查詢並繳費？**

A：請至「**管理設定 → 電子繳費單**」啟用相關功能。啟用後可在學生個人資料中點選「繳費通知」按鈕，發送 APP 推播通知給家長。家長需先完成 APP 綁定。

**Q5：代收人員登入後找不到代收功能？**

A：請確認該人員已在「管理設定 → 代收人員設定」中被加入為代收人員，並且費用單已完成「產生單據」步驟且費用單在有效繳費期間內。

**Q6：費用單的「異常申告」是什麼功能？**

A：若每日銷帳金額與台灣銀行入庫金額不符，可在費用單詳情頁點選「異常申告」，輸入帳戶匯入金額，通知銀行補正差異。
