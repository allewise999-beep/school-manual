# 校園活動報名 操作手冊

校園活動報名模組提供學校開設各類校園活動（如寒暑假育樂營、課後照顧、課後才藝班、補救教學等）與其底下的課程，並支援線上報名、教師推薦、紙本及現場等多種報名方式，搭配教師審核、亂數抽籤、依序錄取等錄取方式，以及併入學費代收、校內收費、教師自收、免費等繳費方式。完成報名錄取後，可線上點名並列印各式名冊、通知單與繳費表單。

本手冊以校務行政系統實際操作畫面說明各功能與操作步驟。

---

## 一、模組簡介

### 1. 模組架構

校園活動報名以「**活動**」為上層、「**課程**」為下層：所有課程都必須開設在某個活動底下。

- **活動**：控制底下每個課程共用的**報名方式、允許報名區間、錄取方式與繳費方式**。
- **課程**：可個別設定該課程的**上課日期與時段、允許報名對象、預計錄取／備取名額、繳費細項與金額**。

因此若有課程的報名／錄取／繳費方式不同，建議開設於不同活動底下。

### 2. 權限說明

| 權限 | 可使用範圍 |
|------|-----------|
| 管理權 | 可新增、刪除、列印及編輯所有活動與課程的相關內容。 |
| 編輯權 | 授課老師可編修自己任課的課程資料，其他非授課老師僅能瀏覽；學生則可看到課程資訊及自己報名的課程。 |
| 停用 | 登入系統時不會看到本模組，也無法做任何操作。 |

### 3. 操作流程

建議依下列流程進行：**參數設定（活動類別／課程類別）→ 新增活動 → 新增課程 → 報名 → 錄取學員 → 課程繳費 → 上課點名 → 表單列印**。

---

## 二、活動管理主畫面

進入模組後即為「以活動檢視」的活動管理主畫面，列出全校各活動（含底下課程數）。可用上方「學年／學期／活動類別」篩選，並由此進入新增活動、課程列表、參數設定與表單列印。

![活動管理主畫面](https://iili.io/CAoewWF.png)

- **① 新增活動**：建立一個新的活動。
- **② 課程列表**：進入該活動底下的課程清單（新增課程、報名、錄取、繳費、點名都在這裡）。
- **③ 參數設定（齒輪圖示）**：設定活動類別、課程類別、外部報名網址與外聘教師。
- **④ 表單列印（印表機圖示）**：列印各式名冊、通知單與繳費表單。
- 上方「課程報名快速查詢」「學生報名快速查詢」可快速查詢報名狀況。

---

## 三、參數設定

點主畫面右上的**齒輪圖示**進入參數設定，內含四個頁籤。

![參數設定](https://iili.io/CAoeefa.png)

- **① 四個頁籤**：
  - **活動類別設定**：新增／維護活動類別（如課後才藝班、課後照顧、補救教學）。
  - **課程類別設定**：新增／維護課程類別。
  - **系統（外部網址）設定**：提供學生／外校學生／外校人士線上報名課程使用的外部網址。
  - **外聘教師管理**：維護外聘（校外）教師資料，供課程指定任課教師時選用。
- **② 新增活動類別**：新增一筆活動類別；如需修改既有類別，請點該類別的「詳細資料」進行修正。

> 建議在新增活動之前，先到參數設定把「活動類別」「課程類別」建立好。

---

## 四、新增活動

於主畫面點「新增活動」進入活動設定表單。單一活動底下所有課程都會套用此活動所設定的報名／錄取／繳費等條件，請審慎填寫。

![新增活動](https://iili.io/CAoek0J.png)

主要欄位：

1. **活動名稱、學年期別、① 活動類別、活動承辦人、電子郵件**：填寫活動基本資料（活動類別需先於參數設定建立）。
2. **活動起訖日**：此活動底下所有課程的上課日都必須落在這個區間內。
3. **報名方式**（四選一，底下課程共用）：
   - **網路線上報名**：公告於外部網頁，供學生／教師／外校人士自行登入報名。
   - **教師推薦**：由教師於系統中直接挑選學員，被挑選者直接錄取。
   - **紙本申請報名**：以紙本收報名，再將名冊登錄系統。
   - **現場報名參加**：現場報名後再登錄系統。
4. **報名時段**：可選「限定報名起訖時間」或「向各課程授課教師報名」（報名期間等同課程上課時間）。
5. **錄取方式**（四選一）：教師審核錄取、依課程抽籤、依員額類別抽籤、依報名順序自動錄取。
6. **繳費方式**（四選一）：併入學費代收、列印單據校內收費、各課程教師自行收費、所有課程免費參加。

> 若有課程的報名／錄取／繳費方式與其他課程不同，請另外開設活動。亦可用「複製」既有活動（含底下課程）的方式，調整後另存為新活動，加速建立。

---

## 五、新增課程

新增活動後，於該活動的「課程列表」點「新增課程」逐項輸入課程內容。課程的報名方式、報名時間、錄取方式與繳費方式由活動設定帶入。

![新增課程](https://iili.io/CAoe6xt.png)

主要欄位：

1. **課程名稱、課程狀態、任課教師、師資說明、課程地點、課程說明、課程網址**。
2. **學員名額**：設定各報名階段的**正取／備取名額**與最低開班人數。
3. **課程對象**：勾選允許報名的年級（並可設定外校學生、本校新生等對象）。
4. **每週上課時段**：逐筆新增上課的星期與起訖時間（上課日須在活動起訖日內）。
5. **報名繳費**：
   - 繳費次數由活動設定帶入；每次繳費最多可設定 5 個項目。
   - **減免設定**：可針對特殊身分設定減免金額，學員身分調整時系統會自動帶出對應金額。
   - **繳費調查**：非必繳項目（如交通費）可勾選，讓報名者自行選擇是否參加；午餐可勾選葷素調查供報名者選填。

> 也可用「複製」既有課程的方式，調整後另存為新課程。

---

## 六、課程內容（報名／錄取／繳費／點名）

於課程列表點各課程的「詳情」進入課程內容，管理者與任課教師皆可在此編修與操作。

![課程列表](https://iili.io/CAoegON.png)

課程列表上方頁籤可切換**報名作業、報名列表、課程製單繳費查詢、單據繳費查詢**等；右上「新增課程」可新增課程，各課程「詳情」進入下列內容頁。

![課程內容](https://iili.io/CAoePWX.png)

課程內容頁以頁籤分為幾個部分：

- **① 學員報名/錄取**：
  - 若有開放外部網頁報名，報名學員會同步呈現於此。
  - **選取學員**：可直接從系統內挑選學生，挑選後直接錄取（不受名額限制）。
  - 依活動設定的錄取方式提供不同按鈕：設「亂數抽籤」會提供抽籤鈕，依名額對所有報名學員抽籤產生錄取名單；設「手動挑選錄取」則由清單勾選錄取。名單確認後按「**錄取名單公佈／錄取完畢**」即鎖定，無法再變更。
- **繳費清單**：設定特殊生減免身分後會帶出對應減免金額；檢視／調整後須按「**暫存設定**」金額才會保留；確認後於期限內「**提交活動製單**」，由活動管理者一併提交出納組統一製單。
- **課程說明 / 上課日期**：即新增課程時輸入的資訊，需修正可在此編修。
- **上課點名**：教師可在此做線上出缺席紀錄。
- **② 選取學員**：直接挑選並錄取學員。
- **③ 課程內容修改**：回到課程設定修改課程資料。

## 七、家長／學生線上查詢報名/錄取課程

開啟參數設定提供的外部報名網址後，家長／學生登入即可做報名與查詢：

- 系統會依登入者身分，將可報名的課程優先置頂，並可用上方篩選快速找到課程。
- 線上報名期間可報名、也可取消報名（放棄錄取）。
- 系統會智慧判斷重複報名／錄取，並依錄取方式不同在不同時機阻擋：
  - **依報名順序錄取**：報名當下即判斷課程時段是否重疊，重疊立即阻擋。
  - **教師審核錄取／亂數抽籤**：學生報名時若時段重疊會提示，須待教師執行錄取後才知是否錄取。

## 八、表單列印

點主畫面右上的**印表機圖示**進入表單列印，左側選擇所需報表，右側設定條件（如活動類別、課程名稱、是否遮蔽姓名等）後列印。

![表單列印](https://iili.io/CAoeQ0G.png)

可列印的表單包含：

| 表單 | 用途 |
|------|------|
| 課程學員名冊 | 列印活動底下課程的學員名冊（可選是否遮蔽姓名）。 |
| 班級學生錄取名冊 | 依年班列出班級學生錄取了哪些課程。 |
| 學員點名清冊 | 列印課程的點名名單。 |
| 課程人數統計 | 呈現課程的報名／錄取人數統計。 |
| 開課通知單 | 列印開課通知（可選是否印地址、加註注意事項）。 |
| 活動課程公告 | 列印課程公告。 |
| 未參加學生名單 | 列出完全沒有參加該活動任何課程的學生。 |
| 繳費清冊 | 以學生或以課程為單位列出費用明細。 |
| 繳費項目分項統計 | 列印繳費狀況、銷帳狀況統計。 |
| 校內收費單據 | 列印校內收費單據。 |
| 班級學生課表 | 依課程日期列印學生課表。 |
| 活動出缺席報表 | 列印活動／課程的出缺席紀錄。 |

> 各階段建議表單：報名前用「活動課程公告」；報名階段用「課程學員名冊、課程人數統計」；報名完成用「班級學生錄取名冊、未參加學生名單」；收費階段用「繳費清冊、學生繳費單」；開課前用「開課通知單、學員點名清冊」。

## 九、課程費用併入學費代收

若活動的繳費方式選擇「**併入學費代收**」，課程費用會併入「學費代收」模組統一由出納組製單，印製繳費聯單，家長可持單到超商、銀行 ATM 等多元繳費。

流程重點：

- 各課程於「課程內容 ＞ 繳費清單」確認金額並「暫存設定」後，在期限內「**提交活動製單**」。
- 活動管理者將活動底下所有課程一併提交至出納組，由出納組於學費代收模組統一製單。
- 單一活動下多項課程費用可合併於單一繳費單據；單一活動也可設定多次繳費，每次的課程、項目、金額皆可獨立設定。

## 十、常見問題

**Q：課程一定要開在活動底下嗎？**

A：是的。所有課程都必須開設在某個活動底下，課程會套用該活動的報名／錄取／繳費方式。

**Q：同一個活動裡，有課程的報名或繳費方式不一樣怎麼辦？**

A：報名方式、錄取方式、繳費方式是「活動」層級、底下課程共用。若某課程方式不同，請另外開一個活動來放它。

**Q：要新開一個跟去年類似的活動／課程，要全部重打嗎？**

A：不用。可用「複製」把之前的活動（含課程）或課程複製出來，調整內容後另存為今年的版本。

**Q：錄取名單按了「錄取完畢／公佈」後還能改嗎？**

A：名單一經公佈會被鎖定，無法再變更，請確認無誤後再公佈。

**Q：學生線上報名時，系統會擋重複報名嗎？**

A：會。依報名順序錄取時，報名當下就判斷時段是否重疊並阻擋；教師審核或抽籤時，會在錄取階段判斷。

**Q：課程費用要併入學費代收，要怎麼送出？**

A：在課程的「繳費清單」確認金額並「暫存設定」，再於期限內「提交活動製單」，由活動管理者一併提交出納組統一製單。

**Q：找不到「標準節次設定」？**

A：現行版本的參數設定為「活動類別設定／課程類別設定／系統(外部網址)設定／外聘教師管理」四個頁籤；課程時段改於「新增課程 ＞ 每週上課時段」逐筆設定上課星期與起訖時間。
