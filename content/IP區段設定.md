# IP 區段設定 操作手冊

IP 區段設定模組用於管控可登入校務行政系統的來源 IP。學校可限定只有特定 IP 網段能自由登入系統，網段以外的 IP 若要登入則需搭配自然人憑證；此外，模組也提供「IP 鎖定清單」，查看因登入失敗而被系統鎖定的 IP，並可將其加入白名單解除鎖定。

本手冊以校務行政系統實際操作畫面說明各功能與操作步驟。

---

## 一、模組簡介

IP 區段設定以兩個分頁提供下列功能：

| 分頁 | 功能說明 |
|------|----------|
| IP 網段設定 | 設定可自由登入系統的 IP 網段；網段以外的 IP 需搭配自然人憑證才能登入。可新增、修改、刪除網段。 |
| IP 鎖定清單 | 查看因登入失敗而被鎖定的 IP，可將其加入白名單以解除鎖定。 |

> 本模組屬系統管理者（管理權）功能，請由具權限的帳號維護。

---

## 二、IP 網段設定

「IP 網段設定」列出目前設定的可登入網段，每筆顯示序號、IP 起始網段～IP 結束網段、啟用狀態、備註說明與最後異動資訊。

- 設定後，**只有落在這些網段內的 IP 可自由登入系統**。
- **網段以外的 IP** 若要登入，需**搭配自然人憑證**使用才可登入。

新增或修改網段的操作如下圖：① 點右上「**新增**」（或某列右側「**修改**」）→ ② 表格出現可編輯欄位，填入 **IP 起始網段～IP 結束網段**、選擇**啟用／停用**、輸入**備註說明**後按「**儲存**」（不存檔可按「取消」）。

![IP 網段設定 — 新增／修改](https://iili.io/CAz3vl1.png)

- **修改**：點該列右側「修改」，欄位變為可編輯，調整後按「儲存」。
- **刪除**：點該列右側「刪除」可刪除該網段設定。

> 注意：設定 IP 網段會限制登入來源，請務必確認自己目前所在的 IP 已包含在允許網段內，以免將自己鎖在系統外。

---

## 三、IP 鎖定清單

當某個 IP 多次登入失敗時，系統會自動將該 IP 鎖定。「IP 鎖定清單」可查看這些被鎖定的 IP，每筆顯示序號、IP、鎖定開始時間與失敗次數。

![IP 鎖定清單](https://iili.io/CAz38UF.png)

- 若確認該 IP 為正常使用者（例如校內同仁因打錯密碼被鎖），可點該列「動作」欄的「**加入白名單**」，即可**解除該 IP 的鎖定狀態**，讓使用者重新登入。
- 上圖測試站目前無被鎖定的 IP，故清單為空；實際有 IP 被鎖定時會逐筆列出並於右側顯示「加入白名單」按鈕。

---

## 四、常見問題

**Q：設了 IP 網段後，外面（家裡）就完全不能登入了嗎？**

A：網段以外的 IP 並非完全不能登入，而是需要**搭配自然人憑證**才能登入；網段內的 IP 則可直接自由登入。

**Q：設定 IP 網段要注意什麼？**

A：請務必確認自己目前使用的 IP 已包含在允許網段內，否則儲存後可能將自己鎖在系統外。

**Q：老師一直打錯密碼被鎖住，怎麼解？**

A：到「IP 鎖定清單」找到該 IP，點「加入白名單」即可解除鎖定，再請老師重新登入。

**Q：誰可以設定 IP 區段？**

A：本模組屬系統管理者（管理權）功能，需以具管理權的帳號登入操作。
