## 一、模組簡介

「【新】場地預約」提供校內場地借用申請、審核及門禁管理功能，上方導覽列分為六大功能（❶ 場地預約、❷ 預約審核、❸ 臨時通行人員、❹ 永久通行人員、❺ 長開管理設定、❻ 場地設定管理）。

![](https://iili.io/CCIwiGf.png)

| 功能 | 說明 | 適用對象 |
|------|------|----------|
| 場地預約 | 查詢及申請場地借用 | 所有人員 |
| 預約審核 | 審核/退件待核准的場地預約申請 | 管理員、場地管理人員 |
| 臨時通行人員 | 設定臨時門禁通行卡 | 管理員 |
| 永久通行人員 | 設定永久門禁通行人員 | 管理員 |
| 長開管理設定 | 設定門禁於特定時間不上鎖 | 管理員（需 RFID 門鎖硬體） |
| 場地設定管理 | 新增及管理場地資料與設定 | 管理員 |

> **權限說明**：管理權可進行全部設定與管理，並可代理預約；編輯權可預約場地，若被授權為場地管理者可進行預約審核；進階瀏覽權/瀏覽權僅能觀看預約狀況。

## 二、場地預約

場地預約提供三個子頁籤切換不同視角（❶ 全部預約、❷ 我的預約、❸ 我的管理），並可新增預約借用（❹）。

![](https://iili.io/CCIws44.png)

{% tabs %}
{% tab title="全部預約" %}

顯示所選場地與日期範圍內的全部預約記錄，管理員可查看所有人員的預約清單（❶ 全部預約頁籤）。

操作步驟：

1. 在「**呈現所有場地**」下拉選單選擇特定場地或全部。
2. 設定查詢日期範圍（預設一週）。
3. 勾選顯示選項：是否隱藏已取消/課務預約/無預約資料。
4. 表格顯示日期、起迄時間、場地名稱、預約事由/申請人/通行人及作業。

{% endtab %}

{% tab title="我的預約" %}

僅顯示目前登入者自己提出的預約申請（❷ 我的預約頁籤），方便追蹤個人預約狀態。

操作步驟：

1. 點選「**我的預約**」頁籤。
2. 可查看自己的待審核、已核准及已退件的預約記錄。
3. 待審核的預約可點「取消」撤回申請。

{% endtab %}

{% tab title="我的管理" %}

以場地管理者身分查看被指定管理的場地預約狀況（❸ 我的管理頁籤）。

![](https://iili.io/CCIwQ3l.png)

操作步驟：

1. 點選「**我的管理**」頁籤。
2. 顯示管理者負責場地的預約申請清單。
3. 管理者可在此直接進行審核或退件。

{% endtab %}
{% endtabs %}

### 新增預約借用

點右上角「**+ 新增預約借用**」（❹）後：

1. 選擇「**預約場地**」。
2. 選擇「**預約時間**」（依場地類型決定選擇方式）：
   - **課表開放預約**：系統帶出課表，勾選非課表時段。
   - **彈性時間預約**：自由選擇起迄時間。
3. 若需**循環預約**，新增兩週預約後第三次點擊新增，系統自動產生後續循環日期。
4. 填寫「**預約事由**」及說明，點「**送出預約**」完成申請。

> 依課表審核/彈性時間審核類型的場地，需等待管理者審核通過才算完成預約。

## 三、預約審核

管理員或場地管理人員可在此審核待核准的預約申請（❶ 預約審核標籤、❷ 核准、❸ 退件）。

![](https://iili.io/CCIwtvS.png)

操作步驟：

1. 點選「**預約審核**」標籤（❶），頁籤上的數字顯示待審核數量。
2. 使用場地篩選及「**待審核/已審核/全部**」過濾顯示清單。
3. 審核操作：
   - 點「**核准**」（❷）通過申請，預約完成。
   - 點「**退件**」（❸）輸入原因後退回，申請人可查看原因並重新申請。
4. 需批次處理時，勾選多筆後點「**批次核准**」或「**批次退件**」。

## 四、場地設定管理

管理員可新增場地、設定場地類型及管理人員（❶ 場地設定管理標籤、❷ 場地匯入、❸ 新增場地）。

{% tabs %}
{% tab title="場地列表" %}

顯示全校所有場地清單，可進行修改、停用或刪除。

![](https://iili.io/CCIwDy7.png)

操作步驟：

1. 點「**場地設定管理**」標籤（❶）。
2. 可使用「**場地匯入**」（❷）批次匯入場地資料；點「**新增場地**」（❸）單筆新增。
3. 場地清單顯示場地名稱、管理人員、借用模式、預約期限及作業。
4. 點「**修改**」可編輯場地資料；點「**停用**」暫停開放；點「**刪除**」移除場地。

{% endtab %}

{% tab title="場地卡機設定" %}

設定各場地的 RFID 卡機門禁資料（需搭配 RFID 硬體使用）。

操作步驟：

1. 點「**場地卡機設定**」頁籤。
2. 選擇場地後設定對應的 RFID 卡機設備。
3. 可進行「**紀錄**」（查看進出紀錄）、「**開門**」（遠端開門）、「**長開**」（設定特定時間不上鎖）等操作。

{% endtab %}
{% endtabs %}

### 新增場地表單

點「**新增場地**」後填寫場地資料（❶ 場地基本資料、❷ 預約類型、❸ 儲存）。

![](https://iili.io/CCIZfja.png)

| 欄位 | 說明 |
|------|------|
| 場地名稱 | 教師預約時顯示的名稱 |
| 位置描述 | 場地所在位置說明 |
| 管理人員 | 負責審核該場地預約的人員 |
| 預約類型 | 不開放／依課表開放／依課表審核／彈性時間開放／彈性時間審核 |
| 納入排課流程 | 勾選後可在【新】課務管理中使用此場地排課 |
| RFID 門鎖啟用 | 設定是否使用刷卡方式進出（需 RFID 硬體） |
| 容納班級數／人數 | 設定同時可使用的容量上限 |

## 五、通行人員管理

{% tabs %}
{% tab title="臨時通行人員" %}

設定特定人員使用臨時通行卡進出門禁場地（❶ 臨時通行人員標籤、❷ 新增臨時通行人員）。

![](https://iili.io/CCINJGj.png)

操作步驟：

1. 點「**臨時通行人員**」標籤（❶）。
2. 可依「類別」篩選已設定的臨時通行卡記錄。
3. 點「**新增臨時通行人員**」（❷），輸入卡片資料及有效期間後儲存。
4. 點「**匯出匯入**」可批次管理臨時通行人員資料。

{% endtab %}

{% tab title="永久通行人員" %}

設定具有長期門禁通行權限的人員（❶ 永久通行人員標籤、❷ 新增永久通行人員）。

![](https://iili.io/CCIN33Q.png)

操作步驟：

1. 點「**永久通行人員**」標籤（❶）。
2. 選擇人員所屬處室後顯示該處室人員名單。
3. 點「**新增永久通行人員**」（❷），選擇人員並設定可通行的場地。
4. 完成設定後該人員可無需預約直接進出指定場地。

{% endtab %}
{% endtabs %}

## 六、長開管理設定

設定門禁在特定時間內不自動上鎖（允許自由進出），需搭配 RFID 門鎖硬體使用。

> ⚠️ 此功能需啟用 RFID 門鎖（在「場地設定管理 → 場地卡機設定」中設定）才能使用。

操作步驟：

1. 點選「**長開管理設定**」標籤。
2. 選擇場地並設定長開時間段（開始時間／結束時間）。
3. 在設定的時間段內，門禁不會自動上鎖，人員可自由進出。
4. 常見用途：早自習、下課時間、放學後等特定開放時段。

## 七、常見問題

**Q1：申請場地後一直顯示「待審核」，什麼時候會核准？**

A：依課表審核或彈性時間審核類型的場地，需等待場地管理人員或管理員手動核准。請確認申請已送出，並聯繫場地管理人員確認審核時程。

**Q2：想預約的時段被課表佔用無法選擇？**

A：依課表開放預約的場地，已被排課使用的時段無法預約。如需在該時段使用，請與教學組長協商，或選擇彈性時間類型的其他場地。

**Q3：循環預約如何操作？**

A：先新增第一次預約，再新增第二次預約（相差一週），系統在第三次點擊新增時，根據前兩次設定的時間規律自動產生後續循環預約日期。

**Q4：預約後想取消，如何操作？**

A：在「場地預約 → 我的預約」中找到該筆預約，點「取消」即可。已通過審核的預約取消後，場地時段會自動釋出。

**Q5：RFID 門禁卡無法感應進門？**

A：請確認：場地卡機設定已啟用 RFID 門鎖、個人通行設定正確（臨時或永久通行人員中有設定）、以及確認時間是否在設定的開放時段內。
