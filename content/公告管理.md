## 一、模組簡介

「【新】公告管理」提供發布、查詢及管理校內公告的功能，支援依處室分類瀏覽、多條件複合查詢，並可設定公告外部網頁顯示方式。

| 功能 | 說明 | 適用對象 |
|------|------|----------|
| 公告列表 | 瀏覽各處室公告，依類別頁籤切換 | 所有人員 |
| 新增公告 | 發布公告，設定對象、類別、等級及附件 | 編輯權以上 |
| 資料檢索 | 多條件複合查詢歷次公告 | 所有人員 |
| 管理設定 | 設定公告外部網頁顯示、類別及等級 | 管理權 |

> **權限說明**：管理權（組長、主任）可使用全部功能；編輯權（導師）可新增、查詢、列印；進階瀏覽權可查詢瀏覽；瀏覽權可瀏覽及列印自己新增的公告。

## 三、新增公告

點選「**+ 新增公告**」按鈕，填寫公告內容並設定對象與類別（❶ 公告對象選擇、❷ 儲存公告）。

![](https://iili.io/CBPxoGV.png)

操作步驟：

1. 點右上角「**+ 新增公告**」按鈕。
2. 選擇「**公告對象**」（❶）：一般人員（所有人可查詢）、機關人員、所屬單位或個人。
3. 設定「**是否置頂**」（是/否）。
4. 選擇「**等級**」（普通等）及「**類別**」（學務處/教務處等）。
5. 填入「**公告主旨**」、「**公告日期**」、「**公告內容**」。
6. 視需要點「**上傳附件**」夾帶檔案。
7. 點「**儲存**」（❷）完成發布，公告立即出現於列表中。

> 公告主旨請精簡明確；日期設定可控制顯示期間，到期後自動下架。

## 四、資料檢索

支援多條件複合查詢公告記錄，可依日期、類別、等級、公告人員及關鍵字篩選（❶ 展開搜尋條件、❷ 設定查詢條件、❸ 執行複合查詢）。

![](https://iili.io/CBPx7wJ.png)

操作步驟：

1. 點「**資料檢索**」（❶）展開搜尋條件面板。
2. 設定查詢條件（❷）：公告日期區間、類別、等級、公告人員、關鍵字。
3. 點「**複合查詢**」（❸）顯示符合條件的公告清單。
4. 點選清單中的公告可查看詳細內容，並可進行列印。

## 五、管理設定

管理員可在設定頁面調整公告外部網頁顯示方式、新增／修改類別及等級（❶ 新增類別、❷ 新增等級）。

![](https://iili.io/CBPxjtf.png)

{% tabs %}
{% tab title="1、公告外部網頁設定" %}


控制外部網頁是否顯示公告及顯示範圍。

操作步驟：

1. 設定「**啟用外部網頁**」（是/否）。
2. 設定顯示公告人員姓名及僅限機關人員公告標題的呈現方式。
3. 設定每頁顯示筆數（建議 10-30 筆）。
4. 複製「**外部網站服務**」連結，提供外部查詢使用。

{% endtab %}
{% tab title="2、類別設定" %}


新增或修改公告類別（如學務處、教務處等）（❶ 新增類別按鈕）。

操作步驟：

1. 點「**新增類別**」（❶）。
2. 輸入類別名稱（以 6 個字為限），設定是否為預設選項及啟用狀態。
3. 點「確定」儲存；現有類別可點「**修改**」調整或「**停用**」。

{% endtab %}
{% tab title="3、等級設定" %}


新增或修改公告重要程度等級（如普通、重要、緊急等）（❷ 新增等級按鈕）。

操作步驟：

1. 點「**新增等級**」（❷）。
2. 輸入等級名稱（以 6 個字為限），設定是否為預設及啟用狀態。
3. 點「確定」儲存；現有等級可點「**修改**」調整。

## 六、常見問題

**Q1：發布公告後在列表中看不到？**

A：請確認公告日期設定是否正確（未到開始日期不會顯示）。另外確認公告對象設定：若設為「個人」則其他人員無法看到。

**Q2：公告無法刪除，只能看到修改？**

A：管理權才有刪除功能；編輯權（導師）只能修改自己發布的公告。若需刪除他人公告，請聯繫有管理權的組長或主任操作。

**Q3：資料檢索找不到特定公告？**

A：請確認查詢日期區間涵蓋公告日期，並確認公告對象設定（「個人」公告僅發布者本人可查詢）。

**Q4：如何讓家長在外部網頁看到公告？**

A：請至「管理設定」中啟用「啟用外部網頁」，在「外部網站服務」中複製外部網頁連結提供給家長使用。

**Q5：為何某些公告無法列印？**

A：已超過公告日期的公告無法執行列印動作。若需列印，請先調整公告日期延長，列印完成後再調回原日期。
